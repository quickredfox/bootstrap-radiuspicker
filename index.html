Dependencies

There are 3 dependencies you must handle yourself before using this component.

  1. jQuery

    <script src="http://code.jquery.com/jquery-1.8.2.min.js" type="text/javascript" charset="utf-8"></script>

  2. Bootstrap by Twitter @mdo & @fat

    • Github https://github.com/twitter/bootstrap
    • CDN http://www.bootstrapcdn.com/ (I never used this)

  3. The Google Map API V3 Javascript library

    Configure it with your own api key and sensor parameter.

Usage

  1. Include the bootstrap-radiuspicker.js code into your page.

    <script src="./bootstrap-radiuspicker/js/bootstrap-radiuspicker.js" type="text/javascript" charset="utf-8"></script>

  2. Build a minimal subset of the bootstrap modal structure.

    • The "trigger" button:

      Must point to the DOM id of the target radiuspicker modal.

      <a href="#radiuspicker" role="button" class="btn" data-toggle="modal">radius picker</a>
    • The "modal":

      The basic modal classes plus and aditional .radiuspicker class on the modal. The .modal-body should only contain an empty div with class .radiuspicker-map and a style attribute to tell Google Maps the dimensions to your map area.

      <div id="radiuspicker" class="modal radiuspicker hide fade">
  <div class="modal-header">
    <button type="button" class="close" data-dismiss="modal" aria-hidden="true">×</button>
    <h3>Radius Picker</h3>
  </div>
  <div class="modal-body">
    <div class="radiuspicker-map" style="width:400px;height:400px"></div>
  </div>
</div>

  3. Initialize the component with an initial latitude, longitude and kilometre radius.

    $('#radiusPicker').radiusPicker( {
	latitude: 51.503226,
	longitude: -0.127051, 
	km: 25
} )

    Listen for changes and handle them how you wish (hint: populate a hidden form input)

    $('#radiusPicker').on('change', function( event, data ){
	/* data ~= { latitude: 51.503226, longitude: -0.127051, km: 32.6456 } */
});

    Optionally you can use miles and metres by replacing km with miles for miles and radius for metres. The change events data objcet will respect that same object structure.
